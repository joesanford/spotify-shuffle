# spotify-shuffle

- `pip install requirements.txt`

- Set environment variables `SPOTIFY_CLIENT_ID` and `SPOTIFY_SECRET`, for Spotify API

- Running:

> ```
> > python sync.py --help
> 
> Usage: sync.py [OPTIONS] COMMAND [ARGS]...
> 
> Options:
>   --help  Show this message and exit.
>
> Commands:
>   shuffle
>   sync
> ```
